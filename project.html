AI Class Project

RESECON 490AI – Artificial Intelligence in Economics

Project Overview

This project explores how AI tools can support managerial economics, focusing on structuring information, analyzing data, and generating decision-focused insights for a business-style research question.

I combine course theory with hands-on experimentation using large language models. The goal is to see when AI is helpful, where it makes mistakes, and how human judgment is still needed to clean up and interpret the results.

The materials below include a short presentation and a written report that summarize the methods, key findings, and reflections on responsible AI use.

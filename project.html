AI Class Project

RESECON 490AI – Artificial Intelligence in Economics

Project Overview

This project applies artificial intelligence tools in a managerial economics context. I use AI to help structure information, explore data, and generate insights for a business-style research question.

The work combines course concepts with practical experimentation: prompting large language models, evaluating their output, and integrating human judgment to improve the final deliverables.

The materials below include a short slide deck and a written report that summarize the approach, findings, and reflections on where AI helped, where it struggled, and how it might be used responsibly in future analysis.
